# SafeShell

Secure shell replacement for AI assistants. Provides full JS/TS power in a sandboxed Deno runtime with pre-configured permissions.

## Features

- **Pre-configured permissions** - No prompts for allowed operations
- **Fine-grained whitelist** - Control external commands (git, docker, etc.) at subcommand and flag level
- **Sandboxed execution** - Leverage Deno's permission system
- **Full JS/TS** - Not limited to shell commands
- **MCP integration** - Use as an MCP server for AI assistants
